Ο [σκουλουκοκομποστήρας](http://www.google.ch/search?q=lombricomposteur&hl=fr&client=safari&rls=en&prmd=imvns&tbm=isch&tbo=u&source=univ&sa=X&ei=4bRlT9_xK8mxhAeZo-msCA&ved=0CGgQsAQ&biw=1401&bih=768) είναι έναν σύστημαν εκτροφής κότσσινων γεωσκουλούκων του είδους [eisenia fedida](http://en.wikipedia.org/wiki/Eisenia_fetida).  

  

  

  

  

  

  

Πως λειτουργεί θα το δείτε δαμαί. Τζιαι γαλλικά να μεν ηξέρετε θα το καταλάβετε που τες εικόνες. 

  

  

Από ότι έμαθα υπάρχουν στην αγοράν τζιαι στην [Ελλάδαν](http://www.antemisaris.gr/product.aspx?iid=2822) (δεν ηξέρω για την Κύπρον). Το πιο αστείον [θα το δείτε δαμαί σε πιθάριν](http://compost-greece.blogspot.com/2009/07/blog-post_4085.html). Αλλά μπορεί κάποιος να κατασκευάσει μόνος του ([κοιτάξετε τον τούτον δαμαί](http://xn--   -2wcslawfa5bl9ajbkogca9gdcdb7u) πως έφτιαξεν έναν, [δαμαί έναν άλλον μοτέλλον](http://www.ecowizz.net/articles/2011/02/le-lombricompostage-comment-fabriquer-un-lombricomposteur/), μπορείτε να το κάμετε με οτιδήποτε μαύρην πλαστικήν κάσιαν που μπαίνει η μια μες την άλλην).

Μετά τες επίμονες αιτήσεις της κόρης μου για κατοικίδιον τζιαι τες φιλοσοφικές συζητήσεις που επακολουθήσαν, εκαταλήξαμεν σε δύο αποφάσεις:

Πρώτη απόφαση: Ότι δεν πιάννουμεν ζώον για να είμαστιν εμείς χαρούμενοι αλλά για να είναι το ζώον χαρούμενον. Η μόνη ηθικά αποδεχτή χαρά που μπορεί να έχουμεν που έναν ζώον είναι η χαρά του να το θωρούμεν ευχαριστημένον ΠΑΝΤΑ, όι ως που να μας περάσει η στρείνα. Ούτε αίσθημαν ιδιοκτησίας, ούτε ψυχολογική ανάγγη, ούτε ζώον εργαλείον, ούτε αντικατάστατον για το πουλουκκούιν. Επειδή τσουρούες που θα ήταν ελεύθερες τζιαι ευχαριστημένες μες το χωράφιν δεν μπορούμεν να έχουμεν διότι δεν ηξέρουμεν ποιός μπορεί να τες φροντίζει άμαν φεύκουμεν για διακοπές, επειδή σσιύλλον δεν μπορούμεν να πάρουμεν για τον ίδιον λόγον τζιαι για το ότι επίσης δεν ξέρουμεν αν θα έχουμεν πάντα όρεξην να τον ηφκάλλουμεν έξω καθημερινά μισήν με μιαν ώραν, επειδή επίσης για τον ίδιον λόγον που δεν μπορούμεν να πάρουμεν τσουρούες δεν μπορούμεν να πάρουμεν ούτε Alpaga, ούτε γαρούθκια, ούτε Γιάκ που κάμνουν πας τα βουνά, αποφασίσαμεν να πάρουμεν σκουλούτζια. Τα σκουλούτζια δεν θέλουν χώρον ούτε περίπατον. Θέλουν σκοτάδιν, σαπισμένα χόρτα τζιαι θερμοκρασίαν 15 με 25 βαθμούς. Μπορείς να φύεις που το σπίτιν 20 με 30 μέρες τζιαι τούτα θα έχουν να τρώσιν που τα παλιά. Αυτά μπορούμεν να τους τα παράσχουμεν ανελλιπώς με αντάλλαγμαν να μας μετατρέπουν τα ποσκούπια μας σε χυμόν ή κομπόστον λίπασμαν.

  

Δευτερή απόφαση:  Να κάμουμεν ότι μπορούμεν να φανταστούμεν μες τον κήπον για να μας προτιμούν πουλλιά, αρκομέλισσες, πασχαλίτσες, σαλαμάντρες, κουφούδες (δεν δακκάννουν οι δικές μας), ελάφκια, βάτραχοι, λιμπούρκα, πεταλλούδες τζιαι ότι άλλον άγριον φκάλλει η φύση μας δακάτω. Η κόρη μου ενδιαφέρεται με την πανίδαν, εγώ με την χλωρίδαν. Έτσι τζιαι αλλιώς πρέπει να έσιεις τζιαι τα θκυο για ναν η τζιαι η πανίδα τζιαι η χλωρίδα ευχαριστημένες.

Ο σκουλουκοκομποστήρας μας δουλεύκει μια χαρά. Ήδη σε 3 μέρες λειτουργείας τα σκουλούτζια μας εγκατασταθήκαν τζιαι εβάλαμεν τους θκυό κιλά φύλλα που τα χορτικά που τρώμεν.

Το περασμένον Σαββατοκυρίακον εκάμαμεν φωλιές για τα πουλιά τζιαι τούτον μαχούμαστιν για ξενοδοχείον εντόμων. Μαχούμαστιν επίσης να σκάψουμεν έναν λούκκον για να κάμουμεν υδροβιότοπον για να έρκουνται βατράχοι, λιβελούλες τζιαι σαλαμάντρες. Ελπίζω να έβρω τον χρόνον να σας κάμω ρεπορτάζ τζιαι για τον σκουλουκοκομποστήραν τζιαι για τα υπόλοιπα της αυλής. Τωρά πάω έξω να αρκέψω το ξενοδοχείον των ζουζουνιών τζιαι το δείλις θα έρτουν κάτι φίλοι να βοηθήσουν να φκάκουμεν τον λούκκον του υδροβιότοπου.
